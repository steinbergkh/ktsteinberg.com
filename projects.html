Website Redesign

Current Website - adpiuofm.wix.com/adpi-1

  • Uses a free, but confusing, domain name from wix
  • Overwhelms user with too many options
  • Photo galleries, one of the main uses of the site, are unintuitive and frustrating to update each year

New Website - umichadpi.com

  • Switched pages from static to dynamic content
  • Uses custom domain
  • Drag and drop image uploader
  • User has ability to add "tags" to images
    • USE CASE: Tagging any photo of the current senior class as "PC-2010", a search for "PC-2010" can then be used to remove images of girls after they have graduated.
    • USE CASE: A photo of girls volunteering for Ronald McDonald House could be tagged as both "sisterhood" and "philanthropy," and when creating photo galleries, the webmaster can decide at a later time which photo gallery it best belongs in.
