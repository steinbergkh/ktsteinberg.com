EECS 370 -- LC2K Instructions


Instruction Type Instruction Name Opcode in binary Action
R-type format add 000 destReg = regA + regB
nand 001 destReg = !(regA & regB) OR ~(regA & regB) (bitwise NOT)
I-type format lw 010 regB = MEM[regA + OFFSET]
sw 011 MEM[regA + OFFSET] = regB
beq 100 if(regA == regB) PC = PC + 1 + OFFSET
J-type format jalr 101 regB = PC + 1; PC = regA
O-type format halt 110 Increment the PC then halt the machine
noop 111 Do nothing
